# U-PROX Concierge

This is a simple Android Application designed to remotely grant access
to entry points managed by U-PROX hardware via the U-PROX Web server.

Compatible with Android 4.4 and up.

## Setup

1. Install the latest APK from GitHub Releases.
2. Open the app and enter your U-PROX Web base URL along with the username and password.

In case you need to change the credentials, simply tap on the gear icon in the App Bar.


## Usage

Press the button for the desired entry point to grant access.
